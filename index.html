GreenQuest

Golf planner tool

GreenQuest is a web project developed during the first-year Project Course in Media Technology at Linnaeus University. The goal is to help golfers discover and plan golf trips across the Småland and Öland regions.

The system integrates APIs for weather, routing and golf club data (Leaflet, OpenRouteService, SMAPI). The focus was on an intuitive user experience, clear information structure and visual balance.

Visit the GreenQuest website

  • Weather integration (OpenWeather API)
  • Interactive map (Leaflet.js)
  • Favorites and route planning features

Student Quiz

Interactive client project for new students

Student Quiz is a group project developed as a client assignment for Linnaeus University. The goal was to create an interactive quiz aimed at helping young people who are considering higher education but are unsure about what to study.

The quiz guides users through a series of personality-based questions, and based on their answers, suggests three educational programs that best match their interests and strengths.

The design and development focused on user engagement, clear visual feedback, and aligning with the university’s official graphic profile to create a fun and trustworthy experience.

Visit the Student Quiz website

  • Personality-based quiz for study guidance
  • Design inspired by Linnaeus University’s brand manual
  • Developed collaboratively in a multidisciplinary student team
  • Built using HTML, CSS and JavaScript
